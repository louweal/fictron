# fictron

This project was submitted to the TRON Season 3 Hackathon on Devpost.

Demo website: [fictron.codesparks.nl](https://fictron.codesparks.nl)

## Setup

```
nvm use 16.13.0
npm install
```

## Run in development mode

```
npm run dev
```

## Run in production mode

```
npm run generate
npm run start
```
